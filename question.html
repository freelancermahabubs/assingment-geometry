No-01 What is DOM and what is the purpose of DOM?

DOM stands for Document Object Model, which is a programming interface for web documents. It represents the structure of an HTML or XML document as a tree-like structure, where each element of the document is represented by a node in the tree.


The purpose of the DOM: is to provide a way for programs and scripts to dynamically access and update the content, structure, and style of a web page. Using the DOM, you can manipulate the elements and properties of a web page using programming languages ​​such as JavaScript, allowing you to create interactive and dynamic web applications.


For example, the DOM specifies that the querySelectorAll method in this code snippet must return a list of all the

elements in the document:


const paragraphs = document.querySelectorAll("p"); // paragraphs[0] is the first

element // paragraphs[1] is the second

element, etc. alert(paragraphs[0].nodeName);

No-02 How will you select HTML elements using DOM? Name the DOM methods?

To select HTML elements using the Document Object Model (DOM), I use a variety of methods provided by the DOM API. Here are some commonly used DOM methods for selecting HTML elements:


getElementById(): Selects an element with a specific ID attribute. getElementsByClassName(): Selects all elements that have a specific class name. getElementsByTagName(): Selects all elements with a specific tag name. querySelector(): Selects the first element that matches a specific CSS selector. querySelectorAll(): Selects all elements that match a specific CSS selector.

No-03 What is event bubble?

Event Bubbling is a concept in the DOM (Document Object Model). It happens when an element receives an event, and that event bubbles up (or you can say is transmitted or propagated) to its parent and ancestor elements in the DOM tree until it gets to the root element.This is the default behavior of events on elements unless you stop the propagation

No-04 What is a callback function and why will you use it?

A callback function is one of the superpowers of JavaScript. It is the way JavaScript passes a function into another function as an argument. The callback function is called in the outer function to execute an action. Arguments in JavaScript are values that are passed to the parameters of a function. They are accessible only in the function they are passed into when that function is called. Think of parameters as variables that are used to hold values in a function.
